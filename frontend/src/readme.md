# Stress Management System - Frontend

A modern React-based frontend for monitoring and managing stress levels using ML predictions.

## Features

- ✅ User Authentication (Login/Signup with JWT)
- ✅ Stress Data Collection Form (8 lifestyle parameters)
- ✅ Real-time Stress Prediction Display
- ✅ Personalized Insights & Recommendations
- ✅ Historical Data Visualization with Charts
- ✅ Voice Assistant for Hands-free Commands
- ✅ Responsive Design with Tailwind CSS

## Voice Commands

- "Check my stress level" - Get current stress analysis
- "Show my history" - View historical stress trends
- "Give recommendations" - Hear personalized suggestions
- "Help" - List available commands

## Tech Stack

- React 18
- React Router v6
- Axios for API calls
- Recharts for data visualization
- Tailwind CSS for styling
- Web Speech API for voice features

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Backend server running (FastAPI)
